Introduction

Bootstrap is a popular open-source front-end framework for building responsive and mobile-first websites.

It includes HTML, CSS, and JS-based design templates for typography, forms, buttons, navigation, and more.

npm install bootstrap

You can add Bootstrap via CDN or install it via npm or yarn.

  • Responsive by default
  • Prebuilt components
  • Consistent design

Components

Bootstrap offers prebuilt components to create interfaces quickly.

Common components include buttons, navbars, cards, and modals.

<button class="btn btn-primary">Click</button>

Just add appropriate classes to activate styling.

  • Buttons
  • Navbars
  • Cards
Utilities

Bootstrap provides utility classes to modify layout, spacing, alignment, and visibility.

These classes are great for quick styling without custom CSS.

.mt-3, .text-center, .d-flex

Utilities help save time and improve consistency.

  • Spacing: m-2, p-3
  • Display: d-none, d-flex
  • Colors: text-danger, bg-success
Customization

You can customize Bootstrap using Sass variables and build tools.

This allows changing colors, spacing, breakpoints, and more.

$primary: #ff5733;

Use the Bootstrap source files and build your own version.

  • Override default variables
  • Create custom components
  • Control file size
